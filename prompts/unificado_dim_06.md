# LÓGICA DE PROCESAMIENTO OBLIGATORIA: "BARRIDO DE CONTRASTE"

Antes de generar el análisis, ejecuta internamente este algoritmo:

1. **Busca el Artefacto (El Dato Duro):** Escanea el texto buscando la descripción física exacta del "proyecto final" o "prototipo" mencionado por el Director o Docente.
2. **Evalúa el Nivel Tecnológico:** Clasifica ese proyecto según el criterio binario (¿Es una manualidad o un producto digital?).
3. **Desactiva el Sesgo de Confirmación:** Si el Director habla de "Industria 4.0" pero el proyecto final descrito es "un manual hecho en Word" o "tejido a mano sin componente digital", clasifícalo como brecha de implementación.

# FOCO DE ANÁLISIS: FORMACIÓN TÉCNICA (EPT) Y TRANSICIÓN LABORAL

**CRITERIO BINARIO DE CLASIFICACIÓN:**

* **MODELO "MANUALIDAD/OFIMÁTICA" (Descartar como ideal):** Proyectos de baja complejidad técnica (artesanía simple, cocina doméstica), uso de TIC solo para "hacer el informe" (Word/PPT), enfoque en tareas repetitivas, ausencia de diseño digital o fabricación asistida.
* **MODELO "INDUSTRIA 4.0/STEAM" (Evidencia Objetivo):** Proyectos que integran Design Thinking, diseño 3D, programación, automatización, comercio electrónico real (Marketplace/Redes), prototipado digital o solución de problemas locales mediante tecnología.

## MAPA DE VARIABLES (Gatilladores de Búsqueda)

Usa estos ejes para filtrar la información. Si el texto menciona estos conceptos, extráelo:

* **EJE A: CALIDAD DEL PROTOTIPO (¿Manualidad o Producto?)**
* *Busca:* La descripción física del "mejor proyecto". ¿Qué hicieron exactamente?
* *Alerta:* Diferenciar entre "hicimos queques/tejidos" (Tradicional) vs. "diseñamos un empaque ecológico con corte láser" (Moderno).


* **EJE B: INTEGRACIÓN DIGITAL (La Capa de Valor)**
* *Busca:* Uso de herramientas avanzadas en el proceso: Diseño 3D, Simulación, E-commerce, Edición Multimedia Pro.
* *Alerta:* Si dicen "usaron computación", verifica si fue solo para escribir el documento (Ofimática) o para crear el producto (Diseño).


* **EJE C: METODOLOGÍA (¿Receta o Diseño?)**
* *Busca:* Evidencia de **Design Thinking** (empatizar, idear, prototipar) o resolución de problemas reales.
* *Alerta:* Instrucción directiva ("hagan esto paso a paso") vs. Innovación estudiantil ("detectaron este problema y crearon esto").


* **EJE D: VISIÓN DE FUTURO (Empleabilidad)**
* *Busca:* Conexión del aprendizaje con estudios superiores (Institutos/CETPRO) o autoempleo real.
